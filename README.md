# 📚 Legado SENAI

Este repositório reúne projetos desenvolvidos ao longo de **2 anos de formação no SENAI**, período em que construí minha base técnica, pratiquei lógica de programação e explorei diferentes ferramentas do desenvolvimento de software.

Aqui está uma coleção de códigos, testes, exercícios e aplicações que marcaram minha jornada de aprendizado.

## 🛠️ Tecnologias e temas explorados

Durante esse período, trabalhei com projetos envolvendo:

- Python
- Django
- JavaScript
- SQL
- Bootstrap
- Flet
- HTML & CSS
- Desenvolvimento Web
- Desenvolvimento Mobile
- Jogos com Pygame
- Estrutura de dados e lógica de programação

## 📂 O que você vai encontrar aqui

Este repositório contém diferentes tipos de materiais, como:

- Aplicações completas
- Sistemas em desenvolvimento
- Exercícios práticos
- Projetos acadêmicos
- Testes de interface
- Estudos de banco de dados
- Experimentos e protótipos

Cada pasta representa uma etapa, desafio ou ideia colocada em prática durante essa fase.

## 🎯 Propósito deste repositório

Mais do que armazenar arquivos, este espaço registra minha evolução como desenvolvedor.

Ele mostra o processo: aprender, errar, ajustar, testar novamente e continuar evoluindo.
